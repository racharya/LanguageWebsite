History

PHP is created by Rasmus Lerdorf in 1995. He was writing some Common Gateway Interface (CGI) in C which he was using to maintain his personal homepage, where he had posted his resume. He wanted to know the amount of traffic his site was receiving, which could not be done by HTML. He extended the CGI to work with web forms and to communicate with databases. He called this implementation "Personal Home Page/Forms Interpreter or PHP/FI".

PHP and HTML

HTML = content- building blocks of web pages. It allows us to put images, text, forms, and so on pieces of content together. It describes to the browser how to display text or other objects in a browser window.

PHP = functionality - makes website "do stuff".

Thus PHP is embedded in HTML to make website dynamic.
