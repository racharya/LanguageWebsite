STEP 1: GETTING PHP INSTALLED

Mac Installation Instructions

Option 1

  • Check if homebrew is installed. In terminal type brew -v Any version should be good. If it isn't installed visit Homebrew to install homebrew
  • Check if PHP is installed. In terminal type php -v Must have at least 7.0. Install/upgrade as needed using homebrew

Option 2

  • Dont want to use homebrew, use MAMP or XAMPP one click solution

Linux Installation Instructions

  • Use XAMPP one click solution

Windows Installation Instructions

  • Use MAMP or XAMPP or WAMP one click solution
