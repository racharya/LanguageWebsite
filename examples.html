Here is an example of the Selection Sort in PHP: selectionSort.php

      <!doctype html>
      <html>
      <body>
      
      <?php
          function selectionSort(array $ary) {
              $n = sizeof($ary);
              for ($i = 0; $i < $n; $i++) {
                  $index = $i;
                  $val = $ary[$i];
                  for ($j = $i + 1; $j < $n; $j++) {
                      if ($ary[$j] < $val) {
                          $index = $j;
                          $val = $ary[$j];
                      }
                  }
                  $ary[$index] = $ary[$i];
                  $ary[$i] = $val;
              }
              return $ary;
          }
      ?>
      
      </body>
      </html>

To run this, you would need first run a web server, such as apache, and go to your local host. However, if you were to just leave the body of this, then you could execute the .php file via the command-line script: 

#!/usr/bin/php [LIST]

Here is an example of the quick Sort in PHP: quick_sort.php

      <!doctype html>
      <html>
      <body>
      <?php
      $unsorted = array(0,5,8,1,9,2,7,6,3,4);
      quick_sort($array)
      {
          // find array size
          $length = count($array);
          // base case test, if array of length 0 then jsut return array to caller
          if($length <= 1){
              return $array;
          }
          else{
          
              // select an item to act as our pivot point, since list is unsorted first position is easiest
              $pivot = $array[0];
              
              // declare our two arrays to act as partitions
              $left = $right = array();
              
              // loop and compare each item in the array to the pivot value, place item in appropriate partition
              for($i = 1; $i < count($array); $i++)
              {
                  if($array[$i] < $pivot){
                      $left[] = $array[$i];
                  }
                  else{
                      $right[] = $array[$i];
                  }
              }
              
              // use recursion to now sort the left and right lists
              return array_merge(quick_sort($left), array($pivot), quick_sort($right));
          }
      }
      ?>
      
      </body>
      </html>

To run this, you would need first run a web server, such as apache, and go to your local host. However, if you were to just leave the body of this, then you could execute the .php file via the command-line script:
