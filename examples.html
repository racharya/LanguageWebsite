Examples of PHP:

Here is an simple example of PHP: HelloWorld.php


      <!doctype html>
      <html>
      <body>
      
      <?php
      echo "Hello World!";
      ?>
      
      <body>
      <html>

To run this, you would need first run a web server, such as apache, and go to your local host. However, if you were to just leave the body of this, then you could execute the .php file via the command-line script: 

#!/usr/bin/php

The output would look like this on the local host web server and from the command above in terminal/console:

Hello World!

Here is an example of PHP using variables, functions, and arithmetic:

      <!doctype html>
      <html>
      <body>
      
      <?php
      $x = 1;
      $y = 2;
      $z = 3;
      
      function letsDoMath() {
          global $x, $y, $z;
          $z = $z + $y + $x;
      }
      
      letsDoMath();
      echo $z;
      ?>
      
      <body>
      <html>

To run this, you would need first run a web server, such as apache, and go to your local host. However, if you were to just leave the body of this, then you could execute the .php file via the command-line script: 

#!/usr/bin/php

The output would look like this on the local host web server and from the command above in terminal/console:

6
